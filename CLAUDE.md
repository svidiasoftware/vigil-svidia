# Vigil — Surveillance Alert Web Dashboard

## Versioning

**IMPORTANT:** Bump the version in `package.json` on every commit with user-facing changes.

- Use semver: `MAJOR.MINOR.PATCH`
- PATCH: bug fixes, small tweaks
- MINOR: new features, UI changes
- MAJOR: breaking changes, major redesigns

Update the version in **both** `package.json` and `src/lib/version.ts`. The version is displayed in the sidebar.
